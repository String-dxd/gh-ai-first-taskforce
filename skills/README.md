# Skills

Claude Code skills installed by `gh ai-first-taskforce setup`.

| Skill | Description |
|---|---|
| [aif-code-review-report](aif-code-review-report/SKILL.md) | Produces a written code review report with findings, code snippets, and actionable suggestions. |
| [aif-create-issue](aif-create-issue/SKILL.md) | Creates a well-structured GitHub issue with complete author and implementer sections for a coding agent to action. |
| [aif-git-hooks-setup](aif-git-hooks-setup/SKILL.md) | Sets up or audits Husky-based pre-commit and pre-push hooks on a JS/TS project. |
| [aif-implement-issue](aif-implement-issue/SKILL.md) | Implements a GitHub issue given an issue number or pasted markdown body. |
| [aif-split-issue](aif-split-issue/SKILL.md) | Decomposes a GitHub issue into atomic child issues, each sized for a single coding-agent PR. |
| [aif-update-npm-dependencies](aif-update-npm-dependencies/SKILL.md) | Audits and updates vulnerable JS/TS dependencies across npm, pnpm, Yarn, and Bun with a 7-day release cooldown. |
